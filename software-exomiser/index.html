Exomiser

The Exomiser is a Java program that functionally annotates and prioritises variants from whole-exome sequencing data starting from a VCF file.

The Exomiser has been developed by our group, the Mouse Informatics Group at the Sanger Institute, and other members of the Monarch initiative.

The website can be found here.

The underlying algorithms and software have been published in several articles (Smedley et al., 2014; Robinson et al., 2014; Zemojtel et al., 2014)
